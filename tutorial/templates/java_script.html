JavaScript Notes

Complete notes covering variables, arrays, functions, DOM, async, OOP, modules and more.

1) What is JavaScript?

JavaScript is a scripting language used for client-side and server-side development. Runs in browsers and Node.js.

2) Variables

let x = 10;      // block scoped
const y = 20;    // cannot reassign
var z = 30;      // function scoped

3) Data Types

let n = 42;          // number
let s = "hello";     // string
let b = true;        // boolean
let u;               // undefined
let o = null;        // null
let obj = {};
let arr = [];

4) Operators

// arithmetic: + - * / %
// comparison: ==, ===, !=, !==, > < >= <=
// logical: &&, ||, !
// ternary:
let result = x>5 ? "big" : "small";

5) Strings

let str = "JavaScript";
str.length;
str.toUpperCase();
str.includes("Script");
`Hello ${str}`;

6) Arrays

let a = [1,2,3];
a.push(4);
a.pop();
a.includes(2);
a.map(x => x*2);
a.filter(x => x>1);
a.reduce((a,b)=>a+b,0);

7) Objects

let user = {name:"John", age:30};
user.name;
user["age"];
user.city = "NY";

8) Functions

function add(a,b){ return a+b; }
const mul = (a,b) => a*b;

9) Loops

for(let i=0;i<5;i++){}
for(let v of [1,2,3]){}
[1,2,3].forEach(x => console.log(x));

10) ES6 Features

// spread
let a2 = [...a,4];
// destructuring
let {name, age} = user;
let [f,s] = a;
